# CodeAlpha FAQ Chatbot

## 📌 Project Description
This is an AI-powered FAQ Chatbot developed using Python, Flask, HTML, CSS, JavaScript, and Pandas. The chatbot answers frequently asked questions by matching user queries with predefined FAQ data stored in a CSV file.

## 🚀 Features
- AI-powered FAQ chatbot
- Fast responses
- User-friendly interface
- Flask backend
- CSV-based knowledge base

## 🛠 Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Pandas

## ▶️ How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the project:
```bash
python app.py
```

3. Open your browser:
```
http://127.0.0.1:5000
```
